# 🔒 Segurança e Privacidade  

A segurança dos usuários é a nossa prioridade.  

Nossos bots **não coletam, armazenam ou compartilham informações pessoais**.  
Todo o processamento ocorre **dentro do próprio Discord**, garantindo que seus dados não sejam expostos a terceiros.  

---

# ✅ Garantias  

- 🔐 Não acessamos mensagens privadas.  
- 📭 Não armazenamos logs de conversas.  
- ⚙️ Só utilizamos permissões estritamente necessárias para o funcionamento dos comandos.  
- 🛡️ Todos os dados (como feedbacks) ficam **apenas no seu servidor** do Discord, sem envio externo.
